# 减肥拍拍乐

一个面向“同时上班、控制体重、偶尔健身”的普通人的非审判型陪伴工具。

## 当前进度

阶段 0 手机端交互原型已于 2026-07-10 验收通过，已完成四个核心入口及各自的可点击回应闭环：

- 我想吃点东西
- 我今天不想练
- 我刚练完，很累
- 没什么，只是来坐坐

现已进入阶段 1“本地优先 MVP”的设计，设计稿见 [`docs/04-阶段1设计.md`](docs/04-阶段1设计.md)。阶段 1 尚未开始实现；当前线上原型的选择仍只保存在页面状态中，刷新后即清空。

阶段 1 只增加本机匿名身份、本地回应、文字记录、时间线、数据导出与清空、PWA 安装和离线体验。数据库以外的后端、登录、真实 AI、图片识别和云同步仍不在范围内。
