# 社区行为准则

《不一样的日记》欢迎不同经验、身体状态和生活方式的人参与。讨论代码和产品时，请保持尊重、具体和善意。

## 我们鼓励

- 针对问题与方案交流，不评价参与者的身体、自律、饮食或运动选择。
- 清楚说明证据、影响和边界，并允许别人提出不同意见。
- 保护真实 Key、照片、日记和其他个人信息。
- 在发现冒犯、骚扰或隐私风险时及时制止并向维护者报告。

## 不接受

- 羞辱、歧视、骚扰、威胁或持续的人身攻击。
- 公开他人的私人信息，或提交真实用户数据作为复现材料。
- 把减重、饮食、运动和身体状态变成道德评价。
- 反复破坏讨论、自动化检查或项目安全边界。

维护者可以编辑、隐藏或删除违反准则的内容，并限制相关参与者继续互动。安全与行为问题请通过仓库的私密漏洞报告入口联系维护者。
